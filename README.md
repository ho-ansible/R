# Ansible role: R
Statistical software

## Requirements
Debian 10, 11.

## Role Variables
+ `R_pkgs` (default: none): list of R packages to install from CRAN
+ `R_sys_pkgs` (default: none): OS (not R) packages to install

## Playbooks
+ `main.yml`: apply role

## Dependencies
None.
